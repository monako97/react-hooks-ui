## 组件

| 组件          | 描述              | 功能                             |
| ------------- | ----------------- | -------------------------------- |
| BrowserMockup | 浏览器样式窗口    | 拖拽/放大/缩小/黑白模式          |
| toast         | 轻提示            | info/success/danger/warn/primary |
| WaveCircle    | 圆形的扩散水波纹  |                                  |
| Markdown      | markdown 渲染组件 | 代码高亮，行，闭合高亮等         |
| MarkdownEdit  | markdown 编辑组件 | 实时渲染 md，内置 emoji 选择器   |
| Button        | 按钮              |                                  |
| BubblyButton  | 扩散按钮          |                                  |
| BackTop       | 返回顶部          |                                  |
| ProgressBar   | 进度条            |                                  |
